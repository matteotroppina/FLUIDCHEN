# Worksheet 1 - Lid Driven Cavity Flow

Build instructions.
```shell
mkdir build && cd build
cmake ..
make
```

Run the code with the Lid Driven Cavity case.
```shell
./fluidchen ../example_cases/LidDrivenCavity/LidDrivenCavity.dat
```
